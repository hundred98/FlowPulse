## 核心亮点

### 1. 完整控制系统

FlowPulse 是一套完整的3D打印控制解决方案：

- **上位机 + 下位机**：Rust高性能上位机 + STM32F407实时固件
- **本地HMI**：支持LVGL、Vue、Solid.js等多种界面方案
- **远程控制**：提供Remote API，支持远程/局域网访问

### 2. 先进运动控制

| 算法支持 | 特点 | 执行时间（test-200.gcode） | 实际速度 |
|---------|------|----------------------------|----------|
| 梯形加减速 | 简单可靠，适合快速打印 | 13.085s | 194.92mm/s |
| S曲线 | 平滑柔顺，降低机械振动 | 13.418s | 190.08mm/s |
| 六点加速 | 精确控制，极致平稳 | 14.256s | 178.91mm/s |

| Gcode文件 | 速度mm/s | 规划时间 | 执行时间 | 平均速度 mm/s |
|--------------|------------|--------------|----------------|----------------|
| test-50.gcode | 50 | 51.10s | 51.08s | 49.94 |
| test-100.gcode | 100 | 25.75s | 25.71s | 99.22 |
| test-200.gcode | 200 | 13.26s | 13.08s | 195.03 |
| test-300.gcode | 300 | 10.81s | 9.04s | 282.19 |

### 3. 高速通信

- **实测57600波特率** 可支持 **400mm/s @ 20000mm/s²加速度**
- 高效串口协议，实时数据传输

### 4. 精密同步

MCU时钟同步，精度达到 **50ns + 1ppm**：

**每10ms段误差≈0.5μs**

**DCF = t_measured / t_expected**：实时漂移补偿

**多种同步模式**：兼顾最高精度和更低开销

### 5. 智能流控

专为实时运动控制设计的可靠通信协议：

- **时间基础流控**：以段运行时间为基础，更精准、更安全
- **重传 + 时间预测**：自动重传 + 延迟补偿优化
- **运控数据按序号传输**：确保数据完整性和顺序
- **独立通道**：非运控数据走独立通道传输

```
┌─────────────────────────────────────────────────┐
│           可靠通信协议                            │
├─────────────────────────────────────────────────┤
│                                                  │
│   运控数据         │  非运控数据                  │
│   (按序号传输)     │  (独立通道)                  │
│         ↓          │         ↓                  │
│   ┌────────────────┴──────────────────┐        │
│   │       流控 + 重传 + 时间预测优化    │        │
│   └────────────────────────────────────┘        │
│                      ↓                          │
│              实时执行                            │
└─────────────────────────────────────────────────┘
```

### 6. 每轴独立配置

每个轴可独立设置速度和加速度参数，灵活适配不同机械结构：

```json
{
  "axes": {
    "x": { "max_speed_mm_per_s": 300, "max_accel": 20000 },
    "y": { "max_speed_mm_per_s": 300, "max_accel": 20000 },
    "z": { "max_speed_mm_per_s": 30, "max_accel": 5000 },
    "e": { "max_speed_mm_per_s": 100, "max_accel": 10000 }
  }
}
```

通过共享内存方式，支持多种本地界面：

```
┌─────────────┐     ┌─────────────┐
│   LVGL      │     │   Vue.js    │
└─────────────┘     └─────────────┘
┌─────────────┐     ┌─────────────┐

```

### 8. 智能圆弧算法

自适应弦长细分，精度与效率的完美平衡：

- **自适应细分**：大圆弧 → 自动少分段（省带宽），小圆弧 → 自动多分段（保精度）
- **整体速度规划**：避免细分后拐点速度不必要降低
- **向心加速度限制**：重要的安全特性，防止过大径向力

```
圆弧质量 = f(精度, 效率, 安全)
FlowPulse: 三者同时最大化
```

### 9. 全面的JSON配置

一个JSON文件掌控一切——告别分散的配置：

| 类别 | 参数 |
|----------|------------|
| **电机** | Step/Dir引脚、微步数、电流、每轴最大速度 |
| **运动** | 速度曲线(梯形/S曲线)、jerk、junction deviation |
| **温度** | PID增益(Kp/Ki/Kd)、传感器类型、ADC引脚 |
| **限位** | 软件限位、归位速度/方向、限位开关位置 |
| **硬件** | 加热器/风扇引脚、探针配置、UART映射 |

**优势**：类型安全、IDE自动完成、Schema验证、易于备份

## 快速开始

### 1. 下载预编译固件

```bash
# 从 Releases 页面下载最新版本
https://github.com/hundred98/FlowPulse/releases

# 运行
.\emb-core-server.exe
```

### 2. 烧录固件

```bash
# 使用 STM32CubeProgrammer
# 1. 连接 ST-Link 编程器
# 2. 加载 firmware.bin (起始地址: 0x08000000)
# 3. 点击 "Start Programming"
```

### 3. 编译上位机

```bash
# 安装 Rust (如果没有)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 克隆
git clone https://github.com/hundred98/FlowPulse.git
cd FlowPulse

# 编译
cargo build --release

# 运行
.\target\release\host.exe "gcodes/test-200.gcode"
```

### 4. 启动桌面UI

```bash
# Windows (MSYS2)
cd LVGL/lvgl-pc-sim
mkdir -p build && cd build
cmake .. -G "MinGW Makefiles"
mingw32-make -j4
./lvgl-pc-sim.exe

# Linux
cd LVGL/lvgl-pc-sim
mkdir -p build && cd build
cmake ..
make -j4
./lvgl-pc-sim
```
